# projeto-airbnb-arquitetura-medallion
Pipeline de dados ponta a ponta utilizando Python, PostgreSQL e Arquitetura Medallion para análise de preços do Airbnb no Rio de Janeiro.

O arquivo listings.csv não foi incluído neste repositório devido ao seu tamanho (82MB), seguindo as boas práticas de não versionar datasets pesados em repositórios de código. Para rodar o projeto, o dataset deve ser obtido via .
